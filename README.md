# Soumik Belel Portfolio

This repository contains a single-page portfolio built with React and Tailwind CSS. The code is bundled into a standalone `index.html` using CDN links for React, ReactDOM, Tailwind, and Babel.

## Usage

1. **Preview locally**: Open `index.html` in a browser (no build step required).
2. **Deploy on GitHub Pages**:
   - Push this folder to a GitHub repository.
   - In the repository settings under *Pages*, select the `main` branch and root folder.
   - The site will be available at `https://<your-username>.github.io/<repo-name>`.

The page includes interactive elements such as a chatbot component (Gemini API) and copy‑email button; some functionality requires an API key.

Feel free to modify personal information, projects, skills, etc. The file uses JSX inside a `<script type="text/babel">` block, so the page compiles on the fly in the browser.
